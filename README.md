# 🛡️ astrbot_plugin_qq_group_manager (QQ群小管家) 

<div align="center">

**✨ QQ群小管家自动化管理解决方案，集成了 LLM 智能互动、潜水清理、宵禁管理与分群定制功能。 ✨**

</div>


## ✨ 核心亮点

**QQ群小管家** 旨在将群管理员从繁琐的日常维护中解放出来。它不仅拥有传统的群管功能，更通过 LLM 赋予了 Bot 独特的“人设”与自然语言处理能力。

### 🔥 LLM 智能反击与互动
- **智能识别辱骂**：当群成员对 Bot 出言不逊时，Bot 会调用 LLM 进行符合人设（毒舌/高冷/傲娇）的反击。
- **动态禁言系统**：自动执行禁言惩罚，时长随机（可配置），且具备 **“累犯翻倍”** 机制。
- **自然语言管理**：管理员可直接对 Bot 说“把 @张三 关进小黑屋10分钟”或“开启全员禁言”，无需死记指令。
- **案底与求情系统**：记录用户违规次数，群友可通过“诚恳道歉”触发 LLM 原谅并清除案底（有冷却时间）。

### 🧹 自动化群维护
- **潜水员清理**：自动检测长时间不发言的成员。
- **人性化机制**：支持清理前发送警告、等级保护、管理员豁免及白名单保护。
- **自动黑名单**：支持“退群自动拉黑”及“被清理自动拉黑”，防止捣乱者反复进群。

### 🌙 秩序与分群定制
- **全局与分群定制**：支持全局默认配置，同时支持用指令为每个群**单独设置**不同的迎新、告别与群规文案。
- **自动宵禁**：指定时间段自动开启全员禁言，守护群友睡眠。
- **入群审批**：支持关键词自动批准入群申请。
- **精美迎新/告别**：支持配置变量和随机抽取本地图片发送。

## 🗣️ LLM 智能互动指南

除了死板的指令，你完全可以像使唤真人管家一样和 Bot 对话（需唤醒或艾特 Bot）：

* **普通群友可问**：
  * *“群里现在有多少人？”* / *“群里有多少潜水的？”* -> 汇报人数或列出潜水名单。
  * *“查一下 @李四 的水表”* -> 汇报该用户的潜水天数、违规案底，并附带毒舌点评。
  * *“群规是什么？”* -> 结合配置动态宣读当前群的群规。
  * *“管家我错了，求求你原谅我吧”* -> 触发求情机制，概率消除违规案底。
* **管理员特权指令**：
  * *“把 @王五 禁言20分钟”* -> 直接执行禁言。
  * *“现在的潜水清理天数是多少？/ 帮我改成15天”* -> 直接通过对话查询或修改底层配置。

## 🛠️ 功能命令一览

| 模块 | 指令 | 说明 |
| :--- | :--- | :--- |
| **🚨 惩罚管理** | `/禁言 [QQ/艾特] [时长]` | 手动禁言（支持 `10m`, `1h` 格式） |
| | `/解禁 [QQ/艾特]` | 解除禁言 |
| | `/赦免 [QQ/艾特]` | **清除违规案底** 并解除禁言 |
| | `/全员禁言 [开启/关闭]` | 手动控制当前群的全员禁言状态 |
| **⚙️ 核心配置** | `/管家监控 [列表/添加/删除] [群号]` | **必须指令**，添加后该群才会启用各项自动化功能 |
| | `/设置群规 [内容]` `(/查看群规)` | 设置本群**专属群规**（覆盖全局默认值） |
| | `/设置欢迎消息 [内容]` `(/查看欢迎消息)` | 设置本群**专属欢迎语** |
| | `/设置退群消息 [内容]` `(/查看退群消息)` | 设置本群**专属告别语** |
| **🛡️ 名单管理** | `/白名单 [列表/添加/删除] [QQ]` | 免死金牌，永远不会被踢 |
| | `/黑名单 [列表/添加/删除] [QQ]` | 禁止入群名单 |
| **🧹 清理工具** | `/清理预览` | 查看当前即将被清理的潜水名单（不执行踢人） |
| | `/清理检查` | 立即手动执行一次清理/警告任务 |
| | `/更新成员` | 强制刷新群成员缓存数据 |
| **❓ 帮助信息** | `/管家帮助` | 获取完整管理指令列表 |

> 💡 **文案变量提示**：
> 设置文案时可使用变量：`{nickname}`(新人昵称)、`{uid}`(新人QQ)、`{username}`(退群者昵称)、`{userid}`(退群者QQ)、`{inactive_days}`(潜水阈值)、`{kick_str}`(踢人/记录文案)。
> 
> 💡 **权限提示**：所有 `/` 开头的管理指令仅限 WebUI 配置中的 `admin_list` 用户使用，且 Bot 需具备群管理员权限。

## ⚙️ 配置说明 (Configuration)

<details>
<summary><strong>点击展开详细配置项说明</strong></summary>

| 配置项 | 描述 | 默认值 |
| :--- | :--- | :--- |
| **启用插件** | 总开关 | `True` |
| **自动宵禁** | 设置开始和结束时间（如 0点-6点） | `关闭` |
| **禁言/惩罚** | 设置 LLM 惩罚的时长范围、倍率上限及案底重置天数 | `1-10分钟` |
| **监控与清理** | 设置潜水阈值（天）、是否自动踢人、是否发送警告、等级保护阈值 | `30天` |
| **入群审批** | 设置自动通过的关键词 | `关闭` |
| **迎新/告别/群规** | 自定义**全局默认**文案，支持上传多张图片供随机发送 | `开启` |
| **安全配置** | 管理员列表、黑/白名单、退群/清理自动拉黑开关 | - |
