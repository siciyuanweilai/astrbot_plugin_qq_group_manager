# 🛡️ astrbot_plugin_qq_group_manager (QQ群小管家) 

<div align="center">

**✨ QQ群小管家自动化管理解决方案，集成了 LLM 智能回怼、潜水清理、宵禁管理与迎新功能。 ✨**

</div>


## ✨ 核心亮点

**QQ群小管家** 旨在将群管理员从繁琐的日常维护中解放出来。它不仅拥有传统的群管功能，更通过 LLM 赋予了 Bot 独特的“性格”。

### 🔥 LLM 智能反击与惩罚
- **智能识别辱骂**：当群成员对 Bot 出言不逊时，Bot 会调用 LLM 进行符合人设（毒舌/高冷）的反击。
- **动态禁言系统**：自动执行禁言惩罚，时长随机（可配置），且具备 **“累犯加倍”** 机制。
- **案底记录**：记录用户的违规次数，屡教不改者将面临更严厉的惩罚。

### 🧹 自动化群维护
- **潜水员清理**：自动检测长时间不发言的成员。
- **人性化机制**：支持清理前发送警告、等级保护、管理员豁免及白名单保护。
- **自动黑名单**：支持“退群自动拉黑”，防止捣乱者反复进群。

### 🌙 秩序与迎新
- **自动宵禁**：指定时间段自动开启全员禁言，守护群友睡眠。
- **入群审批**：支持关键词自动批准入群申请。
- **精美迎新/告别**：支持自定义文案及随机图片发送。

## 🛠️ 功能命令一览

| 模块 | 指令 | 说明 |
| :--- | :--- | :--- |
| **🚨 惩罚管理** | `/禁言 [QQ/艾特] [时长]` | 手动禁言（支持 `10m`, `1h` 格式） |
| | `/解禁 [QQ/艾特]` | 解除禁言 |
| | `/赦免 [QQ/艾特]` | **清除违规案底** 并解除禁言 |
| | `/全员禁言 [开启/关闭]` | 手动控制全员禁言状态 |
| **⚙️ 核心配置** | `/管家监控 [添加/删除] [群号]` | **必须指令**，启用插件功能的群 |
| | `/设置欢迎消息 [内容]` | 自定义当前群的欢迎语 |
| | `/查看欢迎消息` | 预览当前的欢迎配置 |
| **🛡️ 名单管理** | `/白名单 [添加/删除] [QQ]` | 免死金牌，永远不会被踢 |
| | `/黑名单 [添加/删除] [QQ]` | 禁止入群名单 |
| **🧹 清理工具** | `/清理预览` | 查看当前即将被清理的潜水名单 |
| | `/清理检查` | 立即执行一次清理/警告任务 |
| | `/更新成员` | 强制刷新群成员缓存数据 |

> 💡 **提示**：所有管理指令仅限配置中的 `admin_list` 用户使用。

## ⚙️ 配置说明 (Configuration)

<details>
<summary><strong>点击展开详细配置项说明</strong></summary>

| 配置项 | 描述 | 默认值 |
| :--- | :--- | :--- |
| **启用插件** | 总开关 | `True` |
| **自动宵禁** | 设置开始和结束时间（如 0点-6点） | `关闭` |
| **禁言/惩罚** | 设置 LLM 惩罚的时长范围、倍率上限及案底重置天数 | `1-10分钟` |
| **监控与清理** | 设置潜水阈值（天）、是否自动踢人、是否发送警告、等级保护阈值 | `30天` |
| **入群审批** | 设置自动通过的关键词 | `关闭` |
| **迎新/告别** | 自定义文案，上传多张图片供随机发送 | `开启` |
| **安全配置** | 管理员列表、黑/白名单、退群自动拉黑开关 | - |

</details>
